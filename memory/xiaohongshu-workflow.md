# 小红书运营工作流（实用分享在这里）

## 0. 固定定位（长期不变）
- 账号人设：实用资源分享官
- 内容范围：公开免费课程、实用软件、学习/效率资源
- 目标受众：由每条素材自动判断（学生党/职场人/自媒体等）
- 合规边界：不做违规规避、不做侵权资源传播、不做虚假承诺

## 1. 每条内容输入模板（你提供）
你每次只要给这 4 项：
1) 网址
2) 一句话描述
3) 图片（或图片要点）
4) 期望人群（可选，不填则我自动判断）

## 2. 我的固定输出模板（你确认前）
每次固定输出：
1) 受众分析（人群/痛点/触发点）
2) 标题 A/B（抓眼球、可略夸张，但不虚假）
3) 正文（含钩子-价值-行动引导）
4) 标签（8-12 个）
5) 封面文案（短句 + 视觉重点）

## 3. 发布执行规则
- 必须先经你确认，才执行发布
- 默认图文发布（有图片才发）
- 发布后执行评论引导（使用你指定文案）

## 4. 发布后评论引导文案（固定）
7【长按复制，小红书APP见】 3月29日到期，"知识共建共享群"志同道合处 MU5366 :/#b🐯😚😛🍓🐮🐱🐧🌽🐼🥖🐟🥮

## 5. 标准执行口令（你可以直接发）
- “走工作流：<网址> + <描述> + <图片要点>”
- “按工作流改写这条草稿”
- “按工作流发布第N条（已确认）”

## 6. 异常处理
- 链接打不开：先回报并请求替代链接
- 素材不足：先给最小可发版本并标注缺口
- 疑似侵权/违规：暂停发布，给合规改写方案
